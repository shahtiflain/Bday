# Music Files

Place your birthday music file here and name it `birthday-song.mp3`.

## Recommended:
- Use a happy, upbeat birthday song
- Keep file size reasonable (under 5MB recommended)
- Formats supported: MP3, OGG, WAV

## Example sources:
- Royalty-free music from sites like:
  - Pixabay
  - Free Music Archive
  - YouTube Audio Library

## Note:
The music will auto-play when the loader screen finishes. Users can control playback with the music player in the bottom right corner.
